This was made on a M1 macbook. I've not tested on a windows machine.
Using:
  Jogl 2.4.0
  Java "12" 2019-03-19
There is a shellscript to run the project.
  "sh steve_run.sh Hatch" to just compile and run
  "sh run.sh Hatch" with the path file
